# **App Name**: Veloz Commerce

## Core Features:

- Product Showcase: Display product listings with detailed information, including wholesale and retail prices.
- Category Management: Implement product categorization for streamlined browsing and filtering.
- Image Gallery: Showcase of product photos.
- Social Media Links: Configure social media links within the admin panel for centralized management and display in the UI.
- Logo Integration: Showcase the company's logo prominently in the user interface.

## Style Guidelines:

- Primary color: Deep red (#8B0000) to evoke urgency and action.
- Background color: Light gray (#E0E0E0) to create a clean and neutral backdrop.
- Accent color: Black (#000000) for key UI elements, providing a bold contrast.
- Headline font: 'Poppins' (sans-serif) to ensure a modern and clear display.
- Body font: 'PT Sans' (sans-serif), used for longer text.
- Use a dynamic layout that adapts to various devices for optimal viewing experiences. 
- Incorporate subtle animations on page transitions to keep users engaged.
